# 4-3-1: DOMとは何か？

## 🎯 このセクションで学ぶこと

- DOM（Document Object Model）の基本的な概念を理解する
- HTML文書がどのようにツリー構造で表現されるかを説明できる
- JavaScriptがDOMを介してHTMLを操作する仕組みを理解する

## 導入

WebページはHTMLによって構造が定義されていますが、JavaScriptは直接HTMLファイルを編集しているわけではありません。ブラウザがHTMLを解釈して生成する、**DOM（Document Object Model）** というオブジェクトの集まりを操作することで、動的にページの内容を書き換えています。

## 概念の説明

### DOMとは？

DOMは、HTMLやXML文書をオブジェクトのツリー構造として表現したものです。ブラウザはHTMLファイルを読み込むと、それを解析してDOMツリーを構築します。このツリーの各要素（`<html>`, `<body>`, `<div>`, `<p>`など）は**ノード**と呼ばれ、JavaScriptからアクセスできるオブジェクトとして表現されます。

### ツリー構造

DOMは、`document` オブジェクトを頂点とする階層的なツリー構造になっています。各ノードは親子関係や兄弟関係を持ちます。

```text
      document
         |
      <html>
       /   \
    <head>  <body>
     |       /   \
  <title>  <h1>   <p>
```

JavaScriptはこのツリー構造を辿って、特定のノード（HTML要素）にアクセスし、その内容やスタイルを変更したり、新しいノードを追加・削除したりできます。

## 💡 TIP

- Chromeのデベロッパーツールで「Elements」タブを開くと、現在のページのDOMツリーを視覚的に確認できます。

## ✨ まとめ

- DOMはHTML文書をオブジェクトのツリー構造で表現したもの
- ブラウザはHTMLを解析してDOMツリーを構築する
- JavaScriptはDOMを操作することで、Webページを動的に変更する
