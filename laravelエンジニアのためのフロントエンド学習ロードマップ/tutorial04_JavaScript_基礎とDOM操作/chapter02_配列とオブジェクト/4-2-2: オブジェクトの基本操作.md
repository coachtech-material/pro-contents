# 4-2-2: オブジェクトの基本操作

## 🎯 このセクションで学ぶこと

- JavaScriptのオブジェクトの基本的な作成方法とアクセス方法を理解する
- プロパティの追加、更新、削除ができるようになる
- メソッド（オブジェクト内の関数）を定義できるようになる

## 導入

オブジェクトは、キーと値のペアでデータを格納するデータ構造です。関連するデータをまとめて管理するのに便利です。

## 詳細解説

### オブジェクトの作成とアクセス

```javascript
// オブジェクトの作成
const user = {
  name: "John Doe",
  age: 30,
  isAdmin: true,
};

// ドット記法によるアクセス
console.log(user.name); // "John Doe"

// ブラケット記法によるアクセス
console.log(user["age"]); // 30
```

### プロパティの操作

```javascript
// プロパティの追加
user.email = "john.doe@example.com";

// プロパティの更新
user.age = 31;

// プロパティの削除
delete user.isAdmin;

console.log(user);
// { name: 'John Doe', age: 31, email: 'john.doe@example.com' }
```

### メソッド

オブジェクトのプロパティとして関数を定義することができます。これをメソッドと呼びます。

```javascript
const user = {
  name: "John Doe",
  greet: function() {
    console.log("Hello, my name is " + this.name);
  },
};

user.greet(); // "Hello, my name is John Doe"
```

## 💡 TIP

- ブラケット記法は、プロパティ名にスペースやハイフンが含まれる場合や、変数を使ってプロパティにアクセスする場合に便利です。

## ✨ まとめ

- オブジェクトはキーと値のペアでデータを格納する
- プロパティの追加、更新、削除が動的に行える
- オブジェクト内の関数をメソッドと呼ぶ
