# 4-2-3: スプレッド構文と分割代入

## 🎯 このセクションで学ぶこと

- スプレッド構文（`...`）を使って配列やオブジェクトを展開できるようになる
- 分割代入を使って配列やオブジェクトから値を取り出せるようになる

## 導入

スプレッド構文と分割代入は、モダンなJavaScriptで頻繁に使われる非常に便利な機能です。コードをより簡潔で読みやすくすることができます。

## 詳細解説

### スプレッド構文

配列やオブジェクトの前に `...` を付けることで、その要素を展開できます。

```javascript
// 配列の結合
const arr1 = [1, 2];
const arr2 = [3, 4];
const combined = [...arr1, ...arr2];
console.log(combined); // [1, 2, 3, 4]

// オブジェクトのマージ
const obj1 = { a: 1, b: 2 };
const obj2 = { c: 3, d: 4 };
const merged = { ...obj1, ...obj2 };
console.log(merged); // { a: 1, b: 2, c: 3, d: 4 }
```

### 分割代入

配列やオブジェクトから値を取り出して、個別の変数に代入できます。

```javascript
// 配列の分割代入
const numbers = [1, 2, 3];
const [first, second] = numbers;
console.log(first);  // 1
console.log(second); // 2

// オブジェクトの分割代入
const user = { name: "John Doe", age: 30 };
const { name, age } = user;
console.log(name); // "John Doe"
console.log(age);  // 30
```

## 💡 TIP

- オブジェクトのマージで同じキーが存在する場合、後から展開されたオブジェクトの値で上書きされます。

## ✨ まとめ

- スプレッド構文は配列やオブジェクトを展開する
- 分割代入は配列やオブジェクトから値を取り出す
