### 導入

前のセクションで、HTMLがWebページの「骨格」を作ることを学びました。このセクションでは、その骨格を具体的にどのように組み立てていくのか、HTML文書の基本的な構造と、頻繁に登場する主要なタグについて詳しく見ていきましょう。これは、どんなWebページを作る上でも必ず必要となる、まさに「基礎の基礎」です。

### 概念の説明

HTML文書は、入れ子構造（ネスト）になった「要素（Element）」の集まりで構成されています。要素は、`<p>`のような「開始タグ」、`</p>`のような「終了タグ」、そしてその間に挟まれた「内容（Content）」から成り立ちます。そして、すべてのHTML文書には、必ず守らなければならない基本的な構造があります。

#### HTML文書の基本構造

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>文書のタイトル</title>
</head>
<body>
  <!-- ここにページに表示される内容を書く -->
</body>
</html>
```

この構造は、HTMLを書く上での「お作法」のようなものです。一つずつ見ていきましょう。

-   `<!DOCTYPE html>`: この文書がHTML5という最新の規格で書かれていることをブラウザに宣言するものです。必ず一番最初に記述します。
-   `<html>`: すべてのHTML要素を囲む、最も外側の要素です。`lang="ja"`は、このページの主要な言語が日本語であることを示しています。
-   `<head>`: この文書に関する「メタ情報（付加情報）」を格納する部分です。ここに書かれた内容は、直接ページには表示されません。ブラウザのタブに表示されるタイトルや、文字コードの指定、外部CSSファイルの読み込みなどがここに含まれます。
-   `<body>`: このページに**実際に表示されるすべての内容**を格納する部分です。見出し、文章、画像、リンクなど、ユーザーが目にするものはすべてこの`<body>`タグの中に記述します。

### 詳細解説

`<body>`タグの中に記述する、代表的なタグをいくつか紹介します。これらを組み合わせることで、Webページの基本的なコンテンツを作成できます。

#### 1. 見出し (Heading)

見出しは `<h1>` から `<h6>` までの6段階があります。`<h1>` が最も重要度が高く、数字が大きくなるにつれて重要度が低くなります。ページの構造を明確にするために非常に重要です。

-   **コード例:**
    ```html
    <h1>最も大きな見出し</h1>
    <h2>二番目に大きな見出し</h2>
    <h3>三番目に大きな見出し</h3>
    ```
-   **ポイント:** `<h1>` は通常、そのページの主題を示すために1ページに1回だけ使うのが一般的です。

#### 2. 段落 (Paragraph)

文章のまとまり（段落）を表すには `<p>` タグを使います。ブラウザは通常、`<p>` タグの前後に適度な間隔を自動的に設けます。

-   **コード例:**
    ```html
    <p>これは最初の段落です。文章が続きます。</p>
    <p>これは二番目の段落です。新しい段落になります。</p>
    ```

#### 3. リンク (Anchor)

他のページへのリンクを作成するには `<a>` タグを使います。`href` という**属性（Attribute）**に行き先のURLを指定します。

-   **コード例:**
    ```html
    <a href="https://www.google.com">Googleへのリンク</a>
    ```
-   **ポイント:** タグに付加的な情報を与えるものを「属性」と呼びます。`<a>` タグの `href` 属性は、リンク先を指定する重要な属性です。

#### 4. 画像 (Image)

画像を表示するには `<img>` タグを使います。`<img>` タグは終了タグを持たない「空要素」の一つです。`src` 属性に画像のパスを、`alt` 属性に画像が表示できない場合に表示される代替テキストを指定します。

-   **コード例:**
    ```html
    <img src="images/cat.jpg" alt="かわいい猫の画像">
    ```
-   **ポイント:** `alt` 属性は、目の不自由な方が利用するスクリーンリーダーで読み上げられたり、画像が読み込めなかった場合に表示されたりするため、必ず指定するようにしましょう。

#### 5. リスト (List)

項目を列挙するにはリストを使います。順序のないリスト `<ul>` と、順序のあるリスト `<ol>` があります。各項目は `<li>` タグで囲みます。

-   **コード例（順序なし）:**
    ```html
    <ul>
      <li>りんご</li>
      <li>ばなな</li>
      <li>みかん</li>
    </ul>
    ```
-   **コード例（順序あり）:**
    ```html
    <ol>
      <li>まず、お湯を沸かす</li>
      <li>次に、カップに麺を入れる</li>
      <li>最後にお湯を注いで3分待つ</li>
    </ol>
    ```

### 💡 TIP

-   **インデント（字下げ）:** HTMLを書くときは、入れ子構造が分かりやすくなるようにインデント（字下げ）を付ける習慣をつけましょう。コードが格段に読みやすくなります。
-   **コメント:** `<!-- これはコメントです -->` という形式で、HTML内にメモを残すことができます。コメントはブラウザには表示されません。

### ✨ まとめ

-   HTML文書は `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>` という基本構造を持つ。
-   `<head>` にはページのメタ情報を、`<body>` には表示されるコンテンツを記述する。
-   `<h1>`〜`<h6>` で見出しを、`<p>` で段落を定義する。
-   `<a>` タグと `href` 属性でリンクを作成する。
-   `<img>` タグと `src`, `alt` 属性で画像を表示する。
-   `<ul>`（順序なし）と `<ol>`（順序あり）でリストを作成する。
