# Tutorial 2: HTML/CSS基礎

## Chapter 4: Gridによるレイアウト

### Section 2: grid-template-columnsとgrid-template-rows

🖼️ **完成形のイメージ**

このセクションでは、Gridレイアウトの心臓部である`grid-template-columns`と`grid-template-rows`プロパティを学びます。これらのプロパティを使って、Gridコンテナに具体的な列（カラム）と行（ロウ）の構造を定義し、思い通りの格子状レイアウトを作成します。

*ここにスクリーンショットを挿入: (1)3つの均等な幅の列を持つレイアウト (2)サイドバー(1fr)とメインコンテンツ(3fr)のように、比率で幅が指定されたレイアウト*

--- 

🎯 **このセクションで学ぶこと**

このセクションでは、Gridのトラック（列と行）のサイズや数を定義する方法をマスターします。このセクションを終えると、あなたは以下のことができるようになります。

- `grid-template-columns`を使って、グリッドの列の数と幅を定義できるようになる。
- `grid-template-rows`を使って、グリッドの行の数と高さを定義できるようになる。
- `fr`という新しい単位を使い、利用可能なスペースに応じた柔軟な比率ベースのレイアウトを作成できるようになる。
- `repeat()`関数を使って、繰り返しパターンを効率的に記述できるようになる。

--- 

### 導入

`display: grid;`だけでは、ただGridレイアウトを有効にしたに過ぎません。Gridの真価は、これから学ぶ`grid-template-columns`と`grid-template-rows`によって、コンテナ内にどのような「設計図」を描くかを定義するところから始まります。これらのプロパティは、Gridコンテナに「列の幅」と「行の高さ」を教え、アイテムが配置されるべき格子（グリッド）そのものを作り出します。

### 詳細解説

#### `grid-template-columns` (列の定義)

このプロパティは、グリッドの**列の数とそれぞれの幅**を定義します。スペース区切りで値を指定し、値の数がそのまま列の数になります。

- **コード例 (固定幅):**
  ```css
  .container {
    display: grid;
    grid-template-columns: 200px 100px 300px; /* 3つの列を定義 */
  }
  ```
  - **表示結果:** 1列目が200px、2列目が100px、3列目が300pxの幅を持つ3列のグリッドが作成されます。4つ目のアイテムは自動的に次の行の1列目に配置されます。

#### `grid-template-rows` (行の定義)

このプロパティは、グリッドの**行の数とそれぞれの高さ**を定義します。使い方は`grid-template-columns`と全く同じです。

- **コード例 (固定高):**
  ```css
  .container {
    display: grid;
    grid-template-columns: 200px 200px;
    grid-template-rows: 100px 150px; /* 2つの行を定義 */
  }
  ```
  - **表示結果:** 1行目が100px、2行目が150pxの高さを持つグリッドが作成されます。3行目以降のアイテムの高さは、コンテンツに応じて自動的に決まります。

#### 新しい単位 `fr` (fraction / 分数)

Gridレイアウトを非常に強力にしているのが、`fr`という新しい単位です。これは「利用可能なスペースの配分率（分数）」を表します。`fr`を使うことで、コンテナの幅が変わっても、指定した比率を保ったまま列の幅が自動的に調整される、柔軟なレイアウトが簡単に作れます。

- **コード例 (`fr`単位):**
  ```css
  .container {
    display: grid;
    /* 3つの列が 1:2:1 の比率でスペースを分け合う */
    grid-template-columns: 1fr 2fr 1fr;
  }
  ```
  - **表示結果:** コンテナの幅がどれだけであっても、中央の列が両側の列の2倍の幅を常に維持します。

- **固定幅との組み合わせ:**
  ```css
  .container {
    display: grid;
    /* 1列目は200px固定、残りのスペースを2列目がすべて使う */
    grid-template-columns: 200px 1fr;
  }
  ```
  - **表示結果:** サイドバーとメインコンテンツのようなレイアウトが簡単に実現できます。Flexboxの`flex-grow`に似ていますが、より直感的に記述できます。

#### `repeat()` 関数

同じ定義を繰り返す場合、`repeat()`関数を使うとコードを簡潔にできます。

- **コード例 (`repeat()`):**
  ```css
  .container {
    display: grid;
    /* '1fr' を3回繰り返す → 1fr 1fr 1fr と同じ意味 */
    grid-template-columns: repeat(3, 1fr);
  }
  ```
  - **表示結果:** 均等な幅を持つ3列のグリッドが作成されます。12カラムのグリッドシステムなども`repeat(12, 1fr)`と簡単に書けます。

### 💡 TIP

- **`gap`プロパティ:** Flexboxと同様に、Gridでも`gap`プロパティが使えます。`gap: 20px;`と指定すれば、すべての行と列の間に20pxの溝（Gutter）が作られます。`row-gap`と`column-gap`で、行と列の隙間を個別に指定することも可能です。
- **暗黙的なグリッド:** `grid-template-rows`で行の数を指定していなくても、アイテムの数が増えればGridは自動的に新しい行（暗黙的なグリッド）を作成します。この自動作成される行の高さを制御するには`grid-auto-rows`プロパティを使います。

### ✨ まとめ

- `grid-template-columns`で列の数と幅を、`grid-template-rows`で行の数と高さを定義する。
- `fr`単位を使うと、利用可能なスペースを指定した比率で分け合う、柔軟なレイアウトが作れる。
- `repeat()`関数を使うと、同じパターンの繰り返しを簡潔に記述できる。
- `gap`プロパティで、行と列の間の隙間を簡単に設定できる。
