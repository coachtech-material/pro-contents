### 導入

これまでのセクションで、HTMLでページの構造を作り、CSSでレイアウトの骨格（ボックスモデル）を整える方法を学びました。このセクションでは、さらに一歩進んで、ページの「装飾」に焦点を当てます。文字の色やフォント、要素の背景を変えるだけで、Webページの印象は劇的に変わります。ユーザーに情報を分かりやすく伝え、ブランドイメージを表現するための基本的なテクニックをマスターしましょう。

### 詳細解説

#### 1. 色の指定 (`color`)

テキストの色を指定するには、`color`プロパティを使います。色の指定方法にはいくつか種類があります。

-   **カラーネーム:** `red`, `blue`, `black` のように、あらかじめ定義された名前で指定します。手軽ですが、指定できる色の種類は限られます。
-   **HEX（16進数カラーコード）:** `#RRGGBB` の形式で、赤(R)・緑(G)・青(B)の光の三原色をそれぞれ16進数（00〜FF）で指定します。デザインツールなどでも一般的に使われる最もポピュラーな形式です。例: `#000000` (黒), `#FFFFFF` (白), `#FF0000` (赤)。
-   **RGB:** `rgb(R, G, B)` の形式で、赤・緑・青をそれぞれ0〜255の数値で指定します。`rgba(R, G, B, A)` のように `A` (アルファ値: 0〜1) を追加すると、透明度も指定できます。

-   **コード例:**
    ```css
    /* カラーネーム */
    h1 {
      color: darkcyan;
    }

    /* HEX */
    p {
      color: #333333; /* やや薄い黒 */
    }

    /* RGBA（透明度あり） */
    .sub-text {
      color: rgba(0, 0, 0, 0.5); /* 黒の半透明 */
    }
    ```

#### 2. フォントの指定 (`font-*`)

フォント関連のプロパティは、ページの読みやすさとデザイン性を大きく左右します。

-   **`font-family` (フォントの種類):**
    どのフォントで表示するかを指定します。複数のフォントをカンマ区切りで指定すると、ユーザーの環境に最初のフォントがなければ、次のフォントが試されます。最後に `sans-serif` (ゴシック体) や `serif` (明朝体) といった「総称フォントファミリー」を指定するのが一般的です。

-   **`font-size` (文字の大きさ):**
    文字の大きさを指定します。単位には `px` (ピクセル), `rem` (ルート要素に対する相対的な大きさ), `em` (親要素に対する相対的な大きさ) などがあります。最初は `px` で感覚を掴むのが分かりやすいでしょう。

-   **`font-weight` (文字の太さ):**
    文字の太さを指定します。`normal` (標準), `bold` (太字) といったキーワードや、100〜900の数値で指定できます（フォントがその太さに対応している必要があります）。

-   **`line-height` (行の高さ):**
    行と行の間隔を調整します。文字サイズだけを指定すると行間が詰まって見えることがあるため、`1.5` のように単位なしの数値を指定して、文字サイズの1.5倍の高さを確保すると、読みやすさが向上します。

-   **コード例:**
    ```css
    body {
      font-family: "Helvetica Neue", Arial, "Hiragino Kaku Gothic ProN", "Hiragino Sans", Meiryo, sans-serif;
      font-size: 16px;
      line-height: 1.6;
    }

    h2 {
      font-size: 24px;
      font-weight: bold;
    }
    ```

#### 3. 背景の指定 (`background-*`)

要素の背景に色や画像を指定することで、コンテンツを際立たせることができます。

-   **`background-color` (背景色):**
    要素の背景色を指定します。値の指定方法は `color` プロパティと同じです。

-   **`background-image` (背景画像):**
    `url()` 関数を使って、背景に表示したい画像のパスを指定します。デフォルトでは画像が繰り返し表示されます。

-   **背景画像の調整プロパティ:**
    -   `background-repeat`: 画像の繰り返し方法を指定します (`no-repeat`, `repeat-x`, `repeat-y` など)。
    -   `background-position`: 背景画像の表示位置を指定します (`center`, `top right` など)。
    -   `background-size`: 背景画像の大きさを指定します (`cover` は要素全体を覆うように拡大・縮小、`contain` は画像全体が収まるように調整)。

-   **コード例:**
    ```css
    .card {
      background-color: #f0f0f0; /* 薄いグレーの背景色 */
      border: 1px solid #ddd;
      padding: 20px;
    }

    .hero-header {
      background-image: url('../images/hero.jpg');
      background-size: cover;
      background-position: center;
      color: white; /* 背景画像が暗いので文字色を白に */
      padding: 100px 20px;
    }
    ```

### 💡 TIP

-   **Webフォント:** ユーザーの環境に依存しない美しいフォントを使いたい場合は、Google FontsなどのWebフォントサービスを利用するのが一般的です。指定された`<link>`タグをHTMLの`<head>`に追加し、`font-family`でそのフォント名を指定するだけで簡単に導入できます。
-   **色のコントラスト:** 背景色と文字色のコントラスト（明暗差）が低いと、文字が非常に読みにくくなります。特に高齢者や視覚に障がいのある方にとっては深刻な問題です。デザインする際は、十分なコントラスト比を確保するように心がけましょう。Chrome DevToolsなどでチェックすることができます。

### ✨ まとめ

-   テキストの色は `color` プロパティで指定する。
-   色の指定方法には、カラーネーム、HEX、RGB(A)などがある。
-   フォントの種類は `font-family`、大きさは `font-size`、太さは `font-weight` で指定する。
-   行の高さは `line-height` で調整し、読みやすさを確保する。
-   要素の背景には `background-color` で色を、`background-image` で画像を指定できる。
