### Chapter 2 ハンズオン: プロフィールページにスタイルを適用する

🖼️ **完成形のイメージ**

このハンズオンでは、Chapter 1で作成した骨格だけのHTMLファイルに、このチャプターで学んだCSSの知識を総動員してスタイルを適用します。完成すると、以下のようにデザイン性が格段に向上したプロフィールページが出来上がります。

*ここに完成したプロフィールページのスクリーンショットを挿入: 背景色、中央寄せ、フォント、色、余白が適切に設定され、見栄えが良くなったページ*

🎯 **このセクションで学ぶこと**

このハンズオンを終えると、あなたは以下のことができるようになります。

- HTMLファイルに外部CSSファイルを正しくリンクできるようになる。
- セレクタ（クラスセレクタ、IDセレクタ）を使って、特定の要素にスタイルを適用できるようになる。
- ボックスモデル（`margin`, `padding`, `border`）を意識して、レイアウトの余白を調整できるようになる。
- `color`, `font-family`, `background-color` などの装飾プロパティを組み合わせて、ページ全体のデザインを統一できるようになる。

### 🏃 実践: 一緒に作ってみましょう！

Chapter 1で作成した、あの質素な自己紹介ページを、CSSの力で美しく変身させていきましょう。一つ一つのプロパティが、ページの見た目にどう影響を与えるかを体感しながら進めてください。

#### 💭 実装の思考プロセス

CSSでのスタイリングは、大きな部品から小さな部品へ、外側から内側へと進めていくのがセオリーです。いきなり細かい部分の色を変えるのではなく、まずは全体のレイアウトから整えていきます。

**このハンズオンのゴール:**

1.  **準備:** CSSファイルをHTMLに読み込ませる。
2.  **大枠のレイアウト:** ページ全体と、コンテンツを囲むカード部分のレイアウトを中央寄せにする。
3.  **ベースの書式設定:** ページ全体の基本的なフォントや文字色を統一する。
4.  **各パーツの装飾:** 見出し、画像、リンクなどの各要素に個別のスタイルを適用していく。

💡 **ポイント:** CSSは「Cascading（滝のように流れる）」という名前の通り、親要素のスタイルは子要素に継承されます。まず`body`で全体の基準となるスタイルを決め、その後で各要素のスタイルを上書きしていくと、効率的に記述できます。

#### 📝 ステップバイステップで実装

##### Step 1: CSSファイルの作成とリンク

まずは、CSSを記述するためのファイルを作成し、HTMLから読み込みます。

-   **何を考えているか:** 「HTMLとCSSは分離するのが基本。`css`フォルダを作って、その中に`style.css`という名前でファイルを作成しよう。そして、HTMLの`<head>`セクションで`<link>`タグを使って、そのCSSファイルを忘れずに読み込む必要がある。」

1-1. `index.html`と同じ階層に`css`フォルダを作成します。

1-2. `css`フォルダの中に`style.css`という名前で新しいファイルを作成します。

1-3. `index.html`の`<head>`タグ内に、以下の`<link>`タグを追加して`style.css`を読み込みます。

-   **コマンド/コード (index.html):**
    ```html
    <head>
      <meta charset="UTF-8">
      <title>自己紹介</title>
      <link rel="stylesheet" href="css/style.css"> <!-- この行を追加 -->
    </head>
    ```

##### Step 2: 全体のレイアウトとカードのスタイリング

ページ全体の背景色を決め、コンテンツを囲むカードを画面の中央に配置します。

-   **何を考えているか:** 「まず背景を少しグレーにして落ち着いた印象にしたい。次に、プロフィールカード（`div`で囲んだ部分）を中央に寄せたい。`margin: 0 auto;`はブロック要素を中央寄せする常套句だ。カードに`id`を付けて、CSSから狙い撃ちできるようにしよう。」

2-1. `index.html`の、コンテンツ全体を囲む`<div>`に`id="profile-card"`を追加します。

-   **コマンド/コード (index.html):**
    ```html
    <body>
      <div id="profile-card">
        <!-- ...コンテンツ... -->
      </div>
    </body>
    ```

2-2. `style.css`に以下のコードを記述します。

-   **コマンド/コード (style.css):**
    ```css
    /* ページ全体の背景と基本フォント */
    body {
      background-color: #f4f4f4; /* 薄いグレー */
      font-family: sans-serif;
      color: #333;
    }

    /* プロフィールカードのスタイリング */
    #profile-card {
      width: 600px; /* カードの幅を600pxに固定 */
      margin: 40px auto; /* 上下40px、左右は自動で中央寄せ */
      background-color: #ffffff; /* カードの背景は白 */
      padding: 30px; /* カードの内側に30pxの余白 */
      border: 1px solid #ddd; /* 薄いグレーの枠線 */
      border-radius: 8px; /* 角を少し丸くする */
    }
    ```
-   **コマンド解説/コードリーディング:**
    -   `body`セレクタ: ページ全体のスタイルを定義します。
    -   `#profile-card`セレクタ: `id="profile-card"`が指定された要素だけに適用されるスタイルです。
    -   `width: 600px;`: 幅を固定します。
    -   `margin: 40px auto;`: 上下に40pxの余白、左右の余白は`auto`にすることで、親要素の中で中央に配置されます。
    -   `padding: 30px;`: カードの内側、コンテンツとの間に30pxのスペースを作ります。
    -   `border-radius: 8px;`: カードの四隅の角を半径8pxで丸めます。

##### Step 3: プロフィール画像のスタイリング

プロフィール画像を円形に切り抜き、中央に配置します。

-   **何を考えているか:** 「プロフィール画像は丸い方が今風だ。`border-radius: 50%;`で正円にできる。画像はブロック要素じゃないから`margin: 0 auto;`が効かない。だから、画像を`display: block;`でブロック要素に変えてから中央寄せしよう。」

3-1. `index.html`の`<img>`タグに`class="profile-image"`を追加します。

-   **コマンド/コード (index.html):**
    ```html
    <img src="images/profile.png" alt="山田太郎のプロフィール画像" width="150" class="profile-image">
    ```

3-2. `style.css`に以下のコードを追加します。

-   **コマンド/コード (style.css):**
    ```css
    .profile-image {
      display: block; /* ブロック要素に変換 */
      margin: 0 auto 20px; /* 上0, 左右auto(中央), 下20px */
      border-radius: 50%; /* 角を50%丸めて円にする */
    }
    ```

##### Step 4: 見出しとリンクのスタイリング

最後に見出しとSNSリンクのリストのスタイルを整えます。

-   **何を考えているか:** 「名前(`h1`)はもっと目立たせたいから中央寄せに。SNSリスト(`ul`)は、デフォルトの黒丸(・)が邪魔だから消したい。`list-style: none;`で消せる。`padding`も0にしておこう。各リンク(`li`)は横に並べたいな...これは次のFlexboxの章でやろう。今はシンプルに下線を消して、マウスが乗ったら色が変わるようにしよう。」

4-1. `style.css`に以下のコードを追加します。

-   **コマンド/コード (style.css):**
    ```css
    h1 {
      text-align: center; /* テキストを中央寄せ */
      color: #2c3e50;
    }

    h2 {
      border-bottom: 2px solid #3498db; /* 青い下線を追加 */
      padding-bottom: 5px;
      margin-top: 30px;
    }

    ul {
      list-style: none; /* リストの黒丸を削除 */
      padding: 0;
    }

    li a {
      text-decoration: none; /* リンクの下線を削除 */
      color: #3498db; /* リンクの色を青に */
    }

    li a:hover {
      text-decoration: underline; /* マウスが乗ったら下線を表示 */
    }
    ```
-   **コマンド解説/コードリーディング:**
    -   `text-align: center;`: ブロック要素内のテキストやインライン要素を中央に揃えます。
    -   `border-bottom`: 要素の下側にだけ枠線を引きます。
    -   `list-style: none;`: `<ul>`や`<ol>`のマーカー（黒丸や数字）を非表示にします。
    -   `text-decoration: none;`: `<a>`タグのデフォルトの下線を消します。
    -   `:hover`: 「疑似クラス」の一種で、要素にマウスカーソルが乗っている間だけ適用されるスタイルを定義します。

#### ✨ 完成！

お疲れ様でした！`index.html`をブラウザでリロードしてみてください。Chapter 1の時とは見違えるほど、整理されてデザイン性の高いページになったはずです。あなたは今、HTMLの骨格にCSSで魂を吹き込む基本的なスキルを習得しました。
