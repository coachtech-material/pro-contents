# 2-2-2: セレクタと詳細度の基礎

## 🎯 このセクションで学ぶこと

- CSSセレクタの基本的な種類と使い方を理解する
- 詳細度（Specificity）の概念を理解し、スタイルが上書きされるルールを説明できる
- 擬似クラスと擬似要素を使ったスタイリングができるようになる

## 🖼️ 完成イメージ

このセクションでは、特定の要素にスタイルを適用する方法を学びます。例えば、リストの奇数番目と偶数番目で背景色を変えたり、リンクにマウスが乗った時に色を変えたりします。

## 導入

CSSは「どの要素に」「どんなスタイルを」適用するかを指定する言語です。この「どの要素に」を指定するのが**セレクタ**です。セレクタを使いこなすことで、HTML文書内の特定の要素を正確に狙ってスタイルを適用できます。

## 概念の説明

### セレクタとは？

セレクタは、スタイルを適用するHTML要素を選択するためのパターンです。基本的なセレクタには以下の種類があります。

| セレクタの種類 | 説明 | 例 |
|:---|:---|:---|
| **要素セレクタ** | 指定されたHTML要素すべてにスタイルを適用 | `p { color: blue; }` |
| **クラスセレクタ** | 指定されたクラス属性を持つ要素にスタイルを適用 | `.my-class { font-size: 16px; }` |
| **IDセレクタ** | 指定されたID属性を持つ要素にスタイルを適用 | `#my-id { background-color: yellow; }` |

### 詳細度（Specificity）とは？

複数のスタイルが同じ要素に適用される場合、どのスタイルが優先されるかを決定するのが**詳細度**です。詳細度は、セレクタの種類によって計算される点数のようなものです。点数が高いほど優先されます。

- **IDセレクタ**: 100点
- **クラスセレクタ、属性セレクタ、擬似クラス**: 10点
- **要素セレクタ、擬似要素**: 1点

例えば、`#my-id`（100点）は `.my-class`（10点）より優先されます。

## 詳細解説

### 擬似クラスと擬似要素

- **擬似クラス**: 要素の特定の状態を表します。（例: `:hover`, `:focus`, `:nth-child()`）
- **擬似要素**: 要素の特定の部分を表します。（例: `::before`, `::after`, `::first-letter`）

```css
/* リンクにマウスが乗った時のスタイル */
a:hover {
  color: red;
}

/* リストの奇数番目の背景色 */
li:nth-child(odd) {
  background-color: #f0f0f0;
}

/* 段落の最初の文字を大きくする */
p::first-letter {
  font-size: 2em;
}
```

## 💡 TIP

- 詳細度が同じ場合は、後から読み込まれたスタイルが優先されます。
- `!important`を使うと詳細度を無視して強制的にスタイルを適用できますが、多用は避けるべきです。

## 🏃 実践

前のセクションで作成したプロフィールページに、以下のスタイルを追加してみましょう。

1. リンクにマウスが乗ったら色が変わるようにする
2. スキルリストの奇数番目と偶数番目で背景色を変える

## ✨ まとめ

- セレクタはスタイルを適用する要素を選択するためのパターン
- 詳細度はスタイルの優先順位を決めるルール
- 擬似クラスと擬似要素を使うと、より細かいスタイリングが可能になる
