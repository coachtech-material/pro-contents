# 2-2-1: CSSの基本構文と適用方法

## 🎯 このセクションで学ぶこと

- CSSの基本構文（セレクタ、プロパティ、値）を理解する
- CSSをHTMLに適用する3つの方法（インライン、内部、外部）を使い分けられるようになる
- 外部CSSファイルを使った実務的なスタイル管理ができるようになる

## 導入

Chapter 1ではHTMLを使ってWebページの「構造」を作成しました。しかし、HTMLだけでは見た目が味気ないままです。このセクションでは、Webページに「スタイル（見た目）」を与える**CSS（Cascading Style Sheets）**の基本を学びます。

CSSは「HTMLで作った構造に、どのような見た目を適用するか」を指定する言語です。文字の色、背景色、余白、配置など、Webページのデザインに関わるほぼすべてをCSSで制御します。

## CSSの基本構文

CSSは「**セレクタ**」「**プロパティ**」「**値**」の3つの要素で構成されます。

```css
セレクタ {
  プロパティ: 値;
}
```

### 構文の各要素

| 要素 | 役割 | 例 |
|:---|:---|:---|
| **セレクタ** | どの要素にスタイルを適用するかを指定 | `h1`, `.card`, `#header` |
| **プロパティ** | 何を変更するかを指定 | `color`, `font-size`, `margin` |
| **値** | どのように変更するかを指定 | `red`, `16px`, `10px` |

### 実際のコード例

```css
h1 {
  color: blue;
  font-size: 24px;
}
```

このコードを1行ずつ読み解きましょう。

1. `h1` - セレクタ。HTMLの`<h1>`要素を対象にする
2. `{` - 宣言ブロックの開始
3. `color: blue;` - 文字色を青にする
4. `font-size: 24px;` - 文字サイズを24ピクセルにする
5. `}` - 宣言ブロックの終了

> **💡 TIP:** 各宣言の末尾には必ず**セミコロン（`;`）**を付けます。最後の宣言では省略可能ですが、後からプロパティを追加するときにエラーになりやすいため、常に付ける習慣をつけましょう。

## CSSをHTMLに適用する3つの方法

CSSをHTMLに適用する方法は3つあります。それぞれの特徴と使い分けを理解しましょう。

### 1. インラインスタイル（非推奨）

HTML要素の`style`属性に直接CSSを記述する方法です。

```html
<h1 style="color: blue; font-size: 24px;">見出し</h1>
```

**特徴:**
- 特定の要素にのみスタイルを適用できる
- HTMLとCSSが混在し、管理が困難になる
- 同じスタイルを複数の要素に適用する場合、コードが重複する

> **⚠️ 注意:** インラインスタイルは**実務ではほとんど使いません**。JavaScriptで動的にスタイルを変更する場合など、限られた状況でのみ使用します。

### 2. 内部スタイルシート（限定的に使用）

HTMLファイルの`<head>`内に`<style>`タグを使ってCSSを記述する方法です。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>内部スタイルシートの例</title>
  <style>
    h1 {
      color: blue;
      font-size: 24px;
    }
    p {
      color: gray;
      line-height: 1.6;
    }
  </style>
</head>
<body>
  <h1>見出し</h1>
  <p>本文テキスト</p>
</body>
</html>
```

**特徴:**
- 1つのHTMLファイル内でスタイルを管理できる
- 複数のHTMLファイルで同じスタイルを使う場合、コードが重複する
- 小規模なページやプロトタイプ作成時に便利

### 3. 外部スタイルシート（推奨）

CSSを別ファイル（`.css`）に記述し、HTMLから読み込む方法です。

**style.css:**
```css
h1 {
  color: blue;
  font-size: 24px;
}

p {
  color: gray;
  line-height: 1.6;
}
```

**index.html:**
```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>外部スタイルシートの例</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>見出し</h1>
  <p>本文テキスト</p>
</body>
</html>
```

**特徴:**
- HTMLとCSSを分離でき、管理しやすい
- 複数のHTMLファイルで同じCSSファイルを共有できる
- ブラウザがCSSファイルをキャッシュするため、ページ読み込みが高速化される

> **💡 TIP:** 実務では**外部スタイルシート**を使うのが基本です。HTMLは「構造」、CSSは「見た目」と役割を分離することで、保守性が大幅に向上します。

## 3つの方法の比較

| 方法 | 管理のしやすさ | 再利用性 | 実務での使用頻度 |
|:---|:---|:---|:---|
| インラインスタイル | ✕ | ✕ | ほぼ使わない |
| 内部スタイルシート | △ | △ | 限定的 |
| 外部スタイルシート | ◎ | ◎ | **推奨** |

## 🏃 実践: 外部CSSファイルを作成してHTMLに適用する

実際に外部CSSファイルを作成し、HTMLに適用してみましょう。

### Step 1: プロジェクトフォルダを作成

```
css-practice/
├── index.html
└── style.css
```

### Step 2: HTMLファイルを作成

`index.html`を作成し、以下のコードを記述します。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>CSS練習</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>CSSの基本を学ぼう</h1>
  <p>このページでは、CSSの基本構文と適用方法を学びます。</p>
  <p>外部スタイルシートを使って、見た目を整えていきましょう。</p>
</body>
</html>
```

### Step 3: CSSファイルを作成

`style.css`を作成し、以下のコードを記述します。

```css
/* 見出しのスタイル */
h1 {
  color: #333;
  font-size: 28px;
  border-bottom: 2px solid #333;
  padding-bottom: 10px;
}

/* 段落のスタイル */
p {
  color: #666;
  font-size: 16px;
  line-height: 1.8;
}
```

### Step 4: ブラウザで確認

`index.html`をブラウザで開き、スタイルが適用されていることを確認します。

**確認ポイント:**
- 見出しが濃いグレー（#333）で表示されている
- 見出しの下に線が引かれている
- 段落が薄いグレー（#666）で表示されている
- 行間が広めに設定されている

> **💡 TIP:** CSSファイルを変更した後、ブラウザに反映されない場合は、**Ctrl + Shift + R**（Mac: **Cmd + Shift + R**）で強制リロードしてください。ブラウザがキャッシュを使用している可能性があります。

## コメントの書き方

CSSでは`/* */`でコメントを記述できます。コメントはブラウザに無視され、コードの説明やメモとして使用します。

```css
/* これはコメントです */
h1 {
  color: blue; /* 文字色を青に設定 */
}

/*
  複数行の
  コメントも
  書けます
*/
```

## よくある間違いと対処法

### 1. セミコロンの付け忘れ

```css
/* ❌ 間違い */
h1 {
  color: blue
  font-size: 24px;
}

/* ✅ 正しい */
h1 {
  color: blue;
  font-size: 24px;
}
```

### 2. 波括弧の閉じ忘れ

```css
/* ❌ 間違い */
h1 {
  color: blue;

p {
  color: gray;
}

/* ✅ 正しい */
h1 {
  color: blue;
}

p {
  color: gray;
}
```

### 3. ファイルパスの間違い

```html
<!-- ❌ 間違い（ファイル名のスペルミス） -->
<link rel="stylesheet" href="styles.css">

<!-- ✅ 正しい -->
<link rel="stylesheet" href="style.css">
```

> **💡 TIP:** CSSが適用されない場合は、まずブラウザの開発者ツール（F12）の「Console」タブでエラーがないか確認しましょう。ファイルが見つからない場合は「404 Not Found」エラーが表示されます。

## ✨ まとめ

- CSSは「**セレクタ**」「**プロパティ**」「**値**」の3要素で構成される
- CSSをHTMLに適用する方法は3つあるが、実務では**外部スタイルシート**を使用する
- HTMLとCSSを分離することで、コードの**保守性**と**再利用性**が向上する
- CSSが適用されない場合は、セミコロンの付け忘れやファイルパスの間違いを確認する
