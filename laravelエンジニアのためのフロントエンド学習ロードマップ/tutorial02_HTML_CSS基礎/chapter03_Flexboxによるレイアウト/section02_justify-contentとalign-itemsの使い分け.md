# Tutorial 2: HTML/CSS基礎

## Chapter 3: Flexboxによるレイアウト

### Section 2: justify-contentとalign-itemsの使い分け

🖼️ **完成形のイメージ**

このセクションでは、Flexboxの最も強力なプロパティである`justify-content`と`align-items`を学びます。これらを使いこなすことで、アイテムを「主軸方向」と「交差軸方向」に自在に配置し、以下のような多様なレイアウトを実現できるようになります。

*ここにスクリーンショットを挿入: 3つのボックスが、(1)右端揃え、(2)中央揃え、(3)均等間隔、(4)垂直方向に中央揃え、など様々なパターンで配置されている様子を示す*

--- 

🎯 **このセクションで学ぶこと**

このセクションでは、Flexアイテムの揃えをコントロールする2大プロパティ、`justify-content`と`align-items`をマスターします。このセクションを終えると、あなたは以下のことができるようになります。

- `justify-content`を使って、主軸方向（デフォルトでは横方向）の揃え（左揃え、右揃え、中央揃え、均等配置など）を自在に操れるようになる。
- `align-items`を使って、交差軸方向（デフォルトでは縦方向）の揃え（上揃え、下揃え、中央揃えなど）を自在に操れるようになる。
- 「水平・垂直方向の中央揃え」という、これまで非常に難しかったレイアウトをたった数行で実現できるようになる。

--- 

### 導入

前のセクションで、`display: flex;`によってアイテムが横に並ぶことを学びました。しかし、ただ並べるだけでは実用的なレイアウトは作れません。Flexboxの真価は、並べたアイテムを「どのように揃えるか」を簡単に指定できる点にあります。その司令塔となるのが、`justify-content`（主軸方向の揃え）と`align-items`（交差軸方向の揃え）です。この2つのプロパティを理解すれば、あなたのレイアウト能力は飛躍的に向上します。

### 概念の説明

思い出してください、Flexboxには「主軸」と「交差軸」がありましたね。

- **`justify-content`**: **主軸 (Main Axis) 方向**のアイテムの揃え方と間隔を調整します。（デフォルトは横方向）
- **`align-items`**: **交差軸 (Cross Axis) 方向**のアイテムの揃え方を調整します。（デフォルトは縦方向）

この2つは必ず**親要素（Flexコンテナ）**に指定します。子要素（Flexアイテム）に指定しても効果はありません。

### 詳細解説

#### `justify-content` の主要な値

主軸方向（横方向）の揃え方をコントロールします。

| 値 | 説明 | イメージ |
| :--- | :--- | :--- |
| `flex-start` (デフォルト) | コンテナの**始点**に寄せて配置 | `[■■■      ]` |
| `flex-end` | コンテナの**終点**に寄せて配置 | `[      ■■■]` |
| `center` | コンテナの**中央**に配置 | `[   ■■■   ]` |
| `space-between` | 最初と最後のアイテムを両端に配置し、残りのアイテム間の**間隔を均等**にする | `[■   ■   ■]` |
| `space-around` | 全てのアイテムの**周りに均等な間隔**を設ける（両端のアイテムの余白は、アイテム間の余白の半分になる） | `[ ■  ■  ■ ]` |
| `space-evenly` | 全てのアイテムの**間隔と、両端の余白を完全に均等**にする | `[  ■  ■  ■  ]` |

- **コード例 (`space-between`):**
  ```css
  .container {
    display: flex;
    justify-content: space-between; /* これを追加 */
  }
  ```
  - **表示結果:** 最初のアイテムは左端、最後のアイテムは右端に配置され、その間のスペースが均等に分配されます。ヘッダーのロゴとナビゲーションメニューの配置などで非常によく使われます。

#### `align-items` の主要な値

交差軸方向（縦方向）の揃え方をコントロールします。コンテナに高さがないと違いが分かりにくいため、以下の例ではコンテナに高さを指定しています。

| 値 | 説明 | イメージ |
| :--- | :--- | :--- |
| `stretch` (デフォルト) | アイテムの高さをコンテナの高さに合わせて**引き伸ばす** | `[|||]` |
| `flex-start` | コンテナの**始点**（上端）に寄せて配置 | `[▀▀▀]` |
| `flex-end` | コンテナの**終点**（下端）に寄せて配置 | `[▄▄▄]` |
| `center` | コンテナの**中央**に配置 | `[▬▬▬]` |
| `baseline` | アイテム内のテキストの**ベースライン**（下線）を揃える | （文字の大きさが違う場合に効果を発揮） |

- **コード例 (`center`):**
  ```css
  .container {
    display: flex;
    height: 150px; /* 高さを指定 */
    align-items: center; /* これを追加 */
  }
  ```
  - **表示結果:** 高さ150pxのコンテナの中で、アイテムが垂直方向の中央に配置されます。

#### 究極のテクニック: 水平・垂直中央揃え

`justify-content`と`align-items`を組み合わせることで、CSSの長年の課題であった「完全な中央揃え」が、たった3行で実現できます。

- **コード例:**
  ```css
  .container {
    display: flex;
    justify-content: center; /* 水平方向の中央揃え */
    align-items: center;     /* 垂直方向の中央揃え */
    height: 200px; /* コンテナの高さが必要 */
  }
  ```
- **表示結果:** 高さ200pxのコンテナの中で、中のアイテムがど真ん中に配置されます。これはモーダルウィンドウやローディング表示など、あらゆる場面で活用できる必須テクニックです。

### 💡 TIP

- **軸の向きに注意:** `flex-direction: column;`で主軸を垂直にした場合、`justify-content`は「垂直方向」の揃え、`align-items`は「水平方向」の揃えをコントロールするようになります。役割が入れ替わるので注意しましょう。
- **`gap`プロパティ:** `space-between`などでは間隔を直接指定できませんが、最近のブラウザでは`gap`プロパティが使えます。`.container { display: flex; gap: 10px; }`と指定するだけで、アイテム間に10pxの隙間を簡単に作ることができ、非常に便利です。

### ✨ まとめ

- `justify-content`は**主軸方向（横）**の揃えと間隔を、`align-items`は**交差軸方向（縦）**の揃えをコントロールする。
- `justify-content: center;`で水平中央、`align-items: center;`で垂直中央に配置できる。
- この2つを組み合わせることで、完全な中央揃えが簡単に実現できる。
- `space-between`はアイテム間の間隔を均等に分配し、ヘッダーレイアウトなどで頻繁に使われる。
