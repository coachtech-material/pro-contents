# 8-1-1: このチュートリアルで学ぶこと

## 🎯 このチュートリアルのゴール

このチュートリアルでは、Reactの応用的な機能とライブラリの活用方法を学びます。Tutorial 7で学んだ基礎知識をベースに、より実践的なアプリケーション開発に必要なスキルを習得します。

このチュートリアルを完了すると、以下のことができるようになります。

- `useEffect`フックを使って副作用を適切に管理できる
- カスタムフックを作成してロジックを再利用できる
- React Hook FormとZodを使った高度なフォーム管理ができる
- TanStack Queryを使ったサーバー状態管理ができる
- Zustandを使ったグローバル状態管理ができる

## 📚 チュートリアルの構成

このチュートリアルは3つのチャプターで構成されています。

### Chapter 1: useEffectによる副作用の管理

Reactにおける「副作用」の概念を理解し、`useEffect`フックを使って適切に管理する方法を学びます。

- useEffectフックの使い方
- クリーンアップ関数
- React Strict Modeの挙動とクリーンアップの重要性

### Chapter 2: カスタムフック

ロジックを再利用可能な形でカプセル化する「カスタムフック」の作成方法を学びます。

- カスタムフックの作成方法
- ビジネスロジックの分離
- 実務でよく使うカスタムフックの例

### Chapter 3: ライブラリ活用術

実務でよく使われるReactライブラリの使い方を学びます。

- React Hook Formによるフォーム管理
- Zodによるバリデーション
- TanStack Queryによるサーバー状態管理
- Zustandによるグローバル状態管理

## 🛠️ 前提知識

このチュートリアルを始める前に、以下の知識があることを前提としています。

- Reactの基本（コンポーネント、JSX、Props、State）
- TypeScriptの基本的な型定義
- 非同期処理（async/await、fetch API）

## 🚀 さあ、始めましょう！
