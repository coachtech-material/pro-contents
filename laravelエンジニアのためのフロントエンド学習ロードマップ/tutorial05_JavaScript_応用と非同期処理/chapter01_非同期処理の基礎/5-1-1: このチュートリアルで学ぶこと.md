# 5-1-1: このチュートリアルで学ぶこと

## 🎯 このチュートリアルのゴール

このチュートリアルでは、JavaScriptの応用的な機能、特に非同期処理について学びます。非同期処理は、API通信やファイル読み込みなど、時間のかかる処理を効率的に扱うための重要な概念です。

このチュートリアルを完了すると、以下のことができるようになります。

- 同期処理と非同期処理の違いを理解し、適切に使い分けられる
- Promiseを使った非同期処理を記述できる
- async/awaitを使って非同期処理を直感的に書ける
- fetch APIを使ってWeb APIからデータを取得できる
- try/catchを使ってエラーを適切に処理できる

## 💡 なぜ非同期処理を学ぶのか

Webアプリケーションでは、以下のような「時間のかかる処理」が頻繁に発生します。

| 処理 | 例 |
|------|-----|
| API通信 | サーバーからデータを取得する |
| ファイル操作 | 画像をアップロードする |
| タイマー処理 | 一定時間後に処理を実行する |
| データベース操作 | データを保存・取得する |

これらの処理を「同期的」に行うと、処理が完了するまで画面が固まってしまいます。非同期処理を使うことで、時間のかかる処理を「バックグラウンド」で実行しながら、ユーザーは引き続きアプリケーションを操作できます。
