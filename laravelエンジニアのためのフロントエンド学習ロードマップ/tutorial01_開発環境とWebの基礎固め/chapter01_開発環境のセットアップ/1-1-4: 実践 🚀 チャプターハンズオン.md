## 🎯 このセクションで学ぶこと

*   これまでのChapterで学んだ知識（VS Code、ターミナル）を組み合わせ、実際のプロジェクトを始めるための「最初のステップ」を体験します。
*   ターミナルを使ってプロジェクト用のディレクトリを作成し、VS Codeでそのディレクトリを開くという、プロの開発者が毎日行う一連の流れを実践します。
*   簡単なHTMLファイルを作成し、VS Codeの拡張機能「Live Server」を使ってブラウザに表示させることで、開発環境が正しくセットアップされたことを確認します。

### 🏃 実践: 一緒に作ってみましょう！

このチャプターで学んだことを総動員して、記念すべき最初のフロントエンドプロジェクトの雛形を作成し、ブラウザに「Hello, Frontend!」と表示させてみましょう。ここで行う一連の操作は、今後のすべてのチュートリアルで繰り返し行う、基本中の基本となります。

#### 💭 実装の思考プロセス

プロジェクトを始める時、プロのエンジニアはまず「作業場所」を整えます。いきなりファイルを一つ作るのではなく、プロジェクト全体を格納するための専用の「箱」（ディレクトリ）を用意し、そこを拠点として作業を開始します。なぜなら、プロジェクトは通常、多数のファイルやディレクトリで構成されるため、最初にきちんと場所を決めておかないと、後でファイルが散らかってしまい、管理が非常に困難になるからです。

**このハンズオンのゴール:**

1.  **作業場所の確保:** ターミナルを使って、今後の学習で作成するプロジェクトをまとめておくための、大元のディレクトリを作成します。
2.  **プロジェクトの開始:** 作成したディレクトリの中に、今回のハンズオン用の新しいプロジェクトディレクトリを作成し、そこに移動します。
3.  **開発ツールの起動:** VS Codeでプロジェクトディレクトリを開き、開発準備を整えます。
4.  **動作確認:** 簡単なHTMLファイルを作成し、Live Serverでブラウザに表示させることで、環境構築が成功したことを確認します。

💡 **ポイント:** 「場所を決め、拠点を作り、道具を準備し、動作を確認する」という流れは、プログラミングにおけるあらゆる作業の基本です。

#### 📝 ステップバイステップで実装

##### Step 1: プロジェクト用ディレクトリの作成と移動

まずは、今後の学習で作成するすべてのプロジェクトをまとめておくための、大きな箱（ディレクトリ）を用意しましょう。ここでは `projects` という名前にします。

*   **何を考えているか:** 「これからたくさんのプロジェクトを作ることになるから、それらをまとめておく『projects』という名前の引き出しを、自分の部屋（ホームディレクトリ）に作っておこう。そうすれば、後で探すのが楽になる。」

1-1. ターミナルを起動し、`cd` コマンド（引数なし）でホームディレクトリにいることを確認します。

```bash
$ cd
$ pwd
/home/ubuntu
```

1-2. `mkdir` コマンドで `projects` ディレクトリを作成します。

```bash
$ mkdir projects
```

1-3. `ls` コマンドで `projects` ディレクトリが作成されたことを確認します。

```bash
$ ls
Desktop  Documents  Downloads  Music  Pictures  Videos  projects
```

1-4. `cd` コマンドで `projects` ディレクトリに移動します。

```bash
$ cd projects
```

##### Step 2: 今回のハンズオン用プロジェクトの作成

次に、`projects` ディレクトリの中に、今回のハンズオン専用の `hello-frontend` というプロジェクトディレクトリを作成します。

*   **何を考えているか:** 「`projects` の引き出しの中に、最初の練習用の『hello-frontend』という名前の小箱を作ろう。そして、これからの作業はその小箱の中で行うことにしよう。」

2-1. `mkdir` コマンドで `hello-frontend` ディレクトリを作成します。

```bash
$ mkdir hello-frontend
```

2-2. `cd` コマンドで `hello-frontend` ディレクトリに移動します。

```bash
$ cd hello-frontend
```

2-3. `pwd` コマンドで、現在地が `/home/ubuntu/projects/hello-frontend` になっていることを確認します。

```bash
$ pwd
/home/ubuntu/projects/hello-frontend
```

##### Step 3: VS Codeでプロジェクトを開く

いよいよ、この `hello-frontend` ディレクトリをVS Codeで開きます。ターミナルから直接VS Codeを起動する、プロっぽいやり方を試してみましょう。

*   **何を考えているか:** 「作業場所の準備ができたから、いよいよ僕の相棒、VS Codeの出番だ。今いるこの場所（カレントディレクトリ）を、VS Codeで開いて開発を始めよう。」

3-1. ターミナルで、以下のコマンドを入力します。`.` は「カレントディレクトリ（今いるこの場所）」を意味します。

```bash
$ code .
```

##### コマンド解説

*   `code`
    *   → VS Codeを起動するためのコマンドです。（Windowsでインストール時にPATHを通す設定をしたのは、このためです）
*   `.`
    *   → 「カレントディレクトリ」を指す特殊な記号です。つまり、「VS Codeで、今いるこのディレクトリを開いてください」という命令になります。

このコマンドを実行すると、新しいVS Codeのウィンドウが開き、左側のエクスプローラーに `HELLO-FRONTEND` という名前で、空のプロジェクトが開かれているはずです。

##### Step 4: 動作確認用のHTMLファイルを作成

最後に、開発環境が正しく動作するかを確認するために、簡単なHTMLファイルを作成してブラウザに表示させてみましょう。

*   **何を考えているか:** 「環境は整ったけど、本当にちゃんと動くかな？試しに簡単なページを作って、ブラウザで表示できるかテストしてみよう。これが成功すれば、環境構築は完了だ。」

4-1. VS Codeのエクスプローラーで、`HELLO-FRONTEND` の領域にマウスカーソルを合わせ、表示される「新しいファイル」アイコンをクリックします。

4-2. ファイル名を `index.html` として、Enterキーを押します。

4-3. 開かれた `index.html` ファイルに、以下のコードを貼り付けます。

```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Frontend App</title>
</head>
<body>
    <h1>Hello, Frontend!</h1>
</body>
</html>
```

##### Step 5: Live Serverでブラウザ表示

前のセクションでインストールした拡張機能「Live Server」を使って、このHTMLファイルをブラウザで開きます。

5-1. VS Codeのウィンドウ右下にある「Go Live」というボタンをクリックします。

5-2. 自動的にブラウザが起動し、「Hello, Frontend!」と表示されたページが開けば、大成功です！

*   **試してみよう:** VS Codeに戻り、`<h1>` タグの中身を「Hello, Laravel Engineer!」などに書き換えて、ファイルを保存（`[Ctrl] + [S]` または `[⌘] + [S]`）してみてください。ブラウザを手動でリロードしなくても、自動で表示が切り替わるはずです。これがLive Serverの力です！

#### ✨ 完成！

おめでとうございます！あなたは今、プロのエンジニアと同じ手順で開発環境を整え、最初のフロントエンドページを表示させることができました。ターミナルでディレクトリを作り、VS Codeでプロジェクトを開き、Live Serverで動作を確認する。この一連の流れは、あなたのこれからの長い冒険の、記念すべき第一歩です。

### 💡 ヒント

*   ターミナルで `code .` を実行した際に「command not found」と表示される場合、VS Codeのインストール時にPATHが正しく設定されていない可能性があります。その場合は、VS Codeを一度アンインストールし、再度インストール（Windowsの場合は「PATHへの追加」を必ずチェック）し直してみてください。
*   または、VS Codeのコマンドパレット（`[Ctrl] + [Shift] + [P]` または `[⌘] + [Shift] + [P]`）を開き、「`Shell Command: Install 'code' command in PATH`」を実行することでも解決できます。

### 📖 模範解答

このハンズオンを終えた時点で、あなたのPCには以下の構造ができているはずです。

```
/home/ubuntu/  (あなたのホームディレクトリ)
└── projects/
    └── hello-frontend/
        └── index.html
```

そして、ブラウザには `index.html` の内容が表示されています。
