# 9-1-1: このチュートリアルで学ぶこと

## 🎯 このチュートリアルのゴール

このチュートリアルでは、Next.jsを使ったモダンなWebアプリケーション開発を学びます。Next.jsは、Reactをベースにしたフレームワークで、サーバーサイドレンダリング（SSR）、静的サイト生成（SSG）、ファイルベースルーティングなど、本番環境で必要な機能を標準で提供しています。

このチュートリアルを完了すると、以下のことができるようになります。

- Next.js App Routerを使ったアプリケーションを構築できる
- Server ComponentsとClient Componentsを適切に使い分けられる
- 動的ルーティングとデータ取得を実装できる
- レンダリング戦略（SSR、SSG、ISR）を理解し、適切に選択できる
- パフォーマンス最適化のベストプラクティスを適用できる

## 📚 チュートリアルの構成

このチュートリアルは3つのチャプターで構成されています。

### Chapter 1: Next.jsの基本

Next.jsの概要とApp Routerの基本的な使い方を学びます。

- なぜNext.jsなのか
- プロジェクトのセットアップ
- App Routerによるファイルベースルーティング
- Server ComponentsとClient Components

### Chapter 2: ルーティングとデータ取得

Next.jsのルーティングシステムとデータ取得の方法を学びます。

- ファイルベースルーティング
- 動的ルーティング
- データ取得（fetch）
- Loading UIとError UI

### Chapter 3: レンダリングとパフォーマンス

Next.jsのレンダリング戦略とパフォーマンス最適化について学びます。

- レンダリング手法（SSR、SSG、ISR）
- キャッシュ戦略
- パフォーマンス最適化
- Hydration Mismatchエラーの対処法

## 🛠️ 前提知識

このチュートリアルを始める前に、以下の知識があることを前提としています。

- Reactの基本（コンポーネント、Props、State、Hooks）
- TypeScriptの基本的な型定義
- HTML/CSSの基礎知識
- Node.jsとnpmの基本的な使い方
