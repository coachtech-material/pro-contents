4-2-4: 実践 🚀 チャプターハンズオン: ToDoリストのデータ構造を作成する

🎯 このハンズオンの目的

  • 配列とオブジェクトを組み合わせて、実用的なデータ構造を作成する
  • map, filter を使って、データ構造を操作する

課題

ToDoリストのアプリケーションで使われるデータ構造を配列とオブジェクトで表現し、以下の操作を行う関数を作成してください。

  1. ToDoリストのデータ構造を定義する
  2. 各ToDoは、id (数値), text (文字列), completed (真偽値) のプロパティを持つオブジェクトとする

  3. 複数のToDoを格納する配列を作成する

  4. 未完了のToDoだけをフィルタリングする関数を作成する

  5. 指定されたIDのToDoを完了済みにする関数を作成する

💻 解答例

// 1. ToDoリストのデータ構造
let todos = [
  { id: 1, text: "JavaScriptを学ぶ", completed: true },
  { id: 2, text: "Reactを学ぶ", completed: false },
  { id: 3, text: "Next.jsを学ぶ", completed: false },
];

// 2. 未完了のToDoだけをフィルタリングする関数
function getIncompleteTodos(todoList) {
  return todoList.filter(todo => !todo.completed);
}

console.log("未完了のToDo:", getIncompleteTodos(todos));

// 3. 指定されたIDのToDoを完了済みにする関数
function completeTodo(todoList, todoId) {
  return todoList.map(todo => {
    if (todo.id === todoId) {
      return { ...todo, completed: true };
    }
    return todo;
  });
}

todos = completeTodo(todos, 2);
console.log("更新後のToDoリスト:", todos);
