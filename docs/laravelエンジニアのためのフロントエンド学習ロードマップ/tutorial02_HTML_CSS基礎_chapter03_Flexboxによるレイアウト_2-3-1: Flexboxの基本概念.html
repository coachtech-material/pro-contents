Chapter 3: Flexboxによるレイアウト

Section 1: Flexboxの基本概念

🖼️ 完成形のイメージ

このセクションでは、CSSレイアウトの強力なツール「Flexbox」の基本的な仕組みを学びます。Flexboxを使うことで、これまで難しかった「要素の横並び」や「中央揃え」が、驚くほど簡単に実現できることを体感します。

ここにスクリーンショットを挿入: 3つのボックスが綺麗に横に並び、均等な間隔で配置されている様子

🎯 このセクションで学ぶこと

このセクションでは、モダンCSSレイアウトの主流であるFlexboxの基本的な概念と使い方を学びます。このセクションを終えると、あなたは以下のことができるようになります。

  • Flexboxが解決する従来のレイアウトの課題を説明できるようになる。
  • Flexboxの基本構造である「親要素（Flexコンテナ）」と「子要素（Flexアイテム）」の関係を理解する。
  • display: flex; を使って、要素を横並びに配置できるようになる。

導入

これまで学んだCSSでは、要素を縦に並べるのは簡単でしたが、横に並べるのは一苦労でした（floatやinline-blockといった古い手法には多くの問題がありました）。特に、高さの違う要素を綺麗に揃えたり、均等な間隔で配置したりするのは非常に困難でした。Flexboxは、こうしたレイアウトの課題を解決するために登場した、非常に強力で柔軟なレイアウトモジュールです。現代のWeb制作において、Flexboxを使いこなすことは必須のスキルと言えます。

概念の説明

Flexboxを理解する鍵は、「親要素」と「子要素」という2つの役割を意識することです。

  1. 親要素 (Flex Container / フレックスコンテナ):

    • レイアウトをコントロールしたい要素たちの「親」となる要素です。
    • この親要素に display: flex; を指定することで、Flexboxの機能が有効になります。
    • このコンテナが、中のアイテムをどのように並べるかの「司令塔」の役割を果たします。

  2. 子要素 (Flex Item / フレックスアイテム):

    • Flexコンテナの直下にある子要素のことです。
    • これらのアイテムが、コンテナの指示に従って並べられます。
    • 孫要素以降にはFlexboxの能力は直接影響しません。

図解：Flexboxの親子関係

<div class="flex-container"> <!-- 親要素 -->
  <div class="flex-item">アイテム1</div> <!-- 子要素 -->
  <div class="flex-item">アイテム2</div> <!-- 子要素 -->
  <div class="flex-item">アイテム3</div> <!-- 子要素 -->
</div>

.flex-container {
  display: flex; /* この一行で魔法が始まる！ */
}

たったこれだけで、通常は縦に並ぶはずのdiv要素（ブロックレベル要素）が、綺麗に横に並びます。これがFlexboxの基本です。

主軸 (Main Axis) と 交差軸 (Cross Axis)

Flexboxには「軸」という重要な概念があります。

  • 主軸 (Main Axis): Flexアイテムが並ぶ方向の軸です。デフォルトでは水平方向（左から右）です。
  • 交差軸 (Cross Axis): 主軸に直交する軸です。デフォルトでは垂直方向（上から下）です。

ここに主軸と交差軸を示す図を挿入: 横に並んだアイテムに対して、水平方向の矢印が「主軸」、垂直方向の矢印が「交差軸」と示されている

なぜこの「軸」が重要かというと、Flexboxのプロパティは「主軸方向の揃え」と「交差軸方向の揃え」を別々にコントロールするからです。この後のセクションで学ぶjustify-contentは主軸方向、align-itemsは交差軸方向の揃えをそれぞれ担当します。

詳細解説

実際にdisplay: flex;を指定すると、何が起こるのかを見てみましょう。

  • HTML:
    html <div class="container"> <div>アイテム1</div> <div>アイテム2</div> <div>アイテム3</div> </div>

  • CSS (Flexbox適用前):
    css .container div { background-color: lightblue; border: 1px solid blue; padding: 10px; }

    • 表示結果: アイテム1, 2, 3がそれぞれ改行され、縦に並びます。

  • CSS (Flexbox適用後):
    css .container { display: flex; /* 親要素にこれを追加 */ } .container div { background-color: lightblue; border: 1px solid blue; padding: 10px; }

    • 表示結果: アイテム1, 2, 3が改行されず、横に並びます。

flex-directionプロパティ

アイテムを並べる主軸の方向は、flex-directionプロパティで変更することができます。

プロパティ値 説明
row (デフォルト) 左から右へ水平に並べる
row-reverse 右から左へ水平に並べる
column 上から下へ垂直に並べる
column-reverse 下から上へ垂直に並べる
  • コード例 (column):
    css .container { display: flex; flex-direction: column; /* 主軸を垂直方向に */ }
    • 表示結果: アイテムは縦に並びます（通常のブロック要素の並び方と同じに見えますが、Flexboxのコンテキストで制御されている点が異なります）。

💡 TIP

  • Flexboxが効くのは直下の子だけ: Flexコンテナの孫要素には、Flexboxの整列能力は直接働きません。もし孫要素もFlexboxでレイアウトしたい場合は、その親（Flexコンテナの子）にdisplay: flex;を指定して、入れ子にする必要があります。
  • インライン要素もFlexアイテムに: spanなどのインライン要素も、Flexコンテナの直下にあればFlexアイテムとなり、ブロックレベル要素のように振る舞います。

✨ まとめ

  • Flexboxは、要素を柔軟に整列させるためのCSSレイアウトモジュールである。
  • レイアウトしたい要素の親要素にdisplay: flex;を指定して「Flexコンテナ」にする。
  • Flexコンテナの直下の子要素は「Flexアイテム」となり、デフォルトで横に並ぶ。
  • Flexboxには「主軸（アイテムが並ぶ方向）」と「交差軸（それに直交する方向）」という概念がある。
  • flex-directionプロパティで、主軸の方向（並ぶ方向）を変更できる。
