Laravelエンジニアのためのフロントエンド学習ロードマップ

PHP/Laravelエンジニアがフロントエンド開発（JavaScript/TypeScript/React/Next.js）を習得し、2年目レベルのスキルを身につけるための教材です。

Tutorial 1: 開発環境とWebの基礎固め

学習時間: 40時間 | 2チャプター
VS Codeとターミナルの基本操作、Webの仕組み（HTTP、DNS）を学びます。

Tutorial 2: HTML/CSS基礎

学習時間: 50時間 | 4チャプター
HTMLの基本構造、CSSの基本、Flexbox/Gridによるレイアウトを学びます。

Tutorial 3: Tailwind CSS徹底習得

学習時間: 50時間 | 3チャプター
パッケージ管理、Tailwind CSSの導入、レスポンシブデザインを学びます。

Tutorial 4: JavaScript基礎とDOM操作

学習時間: 50時間 | 3チャプター
JavaScriptの基本構文、配列・オブジェクト、DOM操作を学びます。

Tutorial 5: JavaScript応用と非同期処理

学習時間: 50時間 | 3チャプター
非同期処理（Promise, async/await）、fetch API、エラーハンドリングを学びます。

Tutorial 6: TypeScript入門

学習時間: 70時間 | 4チャプター
TypeScriptの基本的な型、開発環境設定、高度な型操作を学びます。

Tutorial 7: React入門

学習時間: 80時間 | 3チャプター
Reactの基本概念、Props、Stateによる状態管理を学びます。

Tutorial 8: React応用

学習時間: 80時間 | 3チャプター
useEffect、カスタムフック、React Hook Form/Zodなどのライブラリを学びます。

Tutorial 9: Next.js

学習時間: 90時間 | 3チャプター
Next.jsの基本、ルーティング、レンダリングとパフォーマンスを学びます。

Tutorial 10: Laravel × Next.js

学習時間: 80時間 | 3チャプター
Laravel SailとNext.jsの連携、API呼び出し、認証機能を学びます。

Tutorial 11: テスト

学習時間: 60時間 | 3チャプター
Vitestによるユニットテスト、PlaywrightによるE2Eテスト、Storybookを学びます。

上級コラム: Webレンダリングの深層理解

学習時間: 60時間（任意） | 0チャプター
ブラウザのレンダリングプロセス、各種レンダリング戦略、RSCを学びます。
