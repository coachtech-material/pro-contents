4-1-4: 条件分岐（if, else if, else）

Chapter 1: JavaScriptの基礎

Section 4: 条件分岐（if, else if, else）

🎯 このセクションで学ぶこと

  • if文を使い、特定の条件が真（true）の場合にのみコードを実行できるようになる。
  • elseを使い、条件が偽（false）の場合の処理を記述できるようになる。
  • else ifを使い、複数の条件を順番に評価し、分岐処理を記述できるようになる。

イントロダクション：プログラムに「判断」させる

前のセクションで学んだ比較演算子と論理演算子は、条件を評価してtrueかfalseを返すものでした。しかし、その結果に基づいて実際のプログラムの動きを変えるには、「もし（if）〜なら、Aの処理をする。そうでなければ、Bの処理をする」という条件分岐の仕組みが必要です。

その中心的な役割を担うのがif文です。if文は、あらゆるプログラミング言語に存在する最も基本的な制御構造の一つであり、プログラムに知性を与える第一歩と言えます。PHPのif文と構文は全く同じなので、Laravelエンジニアにとっては非常に馴染みやすいでしょう。

⚙️ 1. if文：もし〜なら

if文は、指定した条件がtrueの場合にのみ、続く波括弧{}ブロック内のコードを実行します。

基本構文:

if (条件式) {
  // 条件式がtrueの場合に実行される処理
}

例：

const score = 80;

if (score >= 60) {
  console.log("合格です！"); // "合格です！" と表示される
}

const age = 18;
if (age < 20) {
  console.log("未成年です。"); // 表示されない
}
  • score >= 60はtrueなので、ブロック内のconsole.logが実行されます。
  • age < 20はfalseなので、ブロック内の処理はスキップされます。

⚙️ 2. else文：そうでなければ

else文は、if文の条件がfalseだった場合に実行される処理を記述するために使います。if文とセットで使われ、単独で使うことはできません。

基本構文:

if (条件式) {
  // 条件式がtrueの場合に実行される処理
} else {
  // 条件式がfalseの場合に実行される処理
}

例：

const age = 22;

if (age < 20) {
  console.log("未成年です。入場できません。");
} else {
  console.log("成人です。どうぞお入りください。"); // "成人です。どうぞお入りください。" と表示される
}
  • age < 20はfalseなので、ifブロックはスキップされ、elseブロック内のコードが実行されます。

⚙️ 3. else if文：さらに別の条件の場合

3つ以上の条件で分岐させたい場合は、else ifを使います。ifの条件がfalseだった場合に、さらに別の条件で評価を行うことができます。else ifはいくつでも繋げることができます。

基本構文:

if (条件式1) {
  // 条件式1がtrueの場合の処理
} else if (条件式2) {
  // 条件式1がfalseで、条件式2がtrueの場合の処理
} else {
  // すべての条件がfalseだった場合の処理
}

例：

const score = 75;

if (score >= 90) {
  console.log("優");
} else if (score >= 70) {
  console.log("良"); // "良" と表示される
} else if (score >= 50) {
  console.log("可");
} else {
  console.log("不可");
}
  • 評価の順序が重要です。
    1. まずscore >= 90が評価されます → false。
    2. 次にscore >= 70が評価されます → true。このブロックが実行され、console.log("良")が実行されます。
    3. 一度いずれかのブロックが実行されると、それ以降のelse ifやelseはすべて無視されます。

💡 三項演算子：シンプルなif-elseのショートカット

if-elseの処理が非常にシンプルな場合、三項演算子（? :）を使うと、コードを1行で簡潔に書くことができます。

基本構文:

条件式 ? (trueの場合の値) : (falseの場合の値)

例：

const age = 20;

// if-else文の場合
let message;
if (age >= 20) {
  message = "成人";
} else {
  message = "未成年";
}

// 三項演算子の場合
const message = age >= 20 ? "成人" : "未成年";

console.log(message); // "成人"

三項演算子は、特にReactなどのフレームワークで、コンポーネントの表示を条件によって切り替える際などによく使われます。ただし、処理が複雑になる場合は、可読性の観点から通常のif-else文を使う方が適切です。

✨ まとめ

  • ifは、条件がtrueのときに処理を実行する、条件分岐の基本である。
  • elseは、ifの条件がfalseだった場合の「それ以外」の処理を記述する。
  • else ifは、複数の条件を順番に評価し、複雑な分岐を実現する。
  • シンプルなif-elseは、三項演算子（? :）で簡潔に記述できる。
