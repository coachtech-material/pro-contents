Chapter 3: Flexboxによるレイアウト

Chapter 3 ハンズオン: Flexboxでプロフィールページをモダンなレイアウトに改良する

🖼️ 完成形のイメージ

このハンズオンでは、Chapter 2でスタイリングしたプロフィールページにFlexboxを導入し、よりモダンで柔軟なレイアウトにアップグレードします。特に、SNSリンクがただ縦に並んでいるだけだった部分を、綺麗に横並びで中央に配置します。

ここに完成したプロフィールページのスクリーンショットを挿入: ヘッダー部分がFlexboxで左右に配置され、SNSリンクのリストが横並びになり、カードフッターも左右に要素が配置されている様子

🎯 このセクションで学ぶこと

このハンズオンを終えると、あなたは以下のことができるようになります。

  • 既存のレイアウトにFlexboxを段階的に導入し、コンポーネントを改良できるようになる。
  • justify-content と align-items を使って、実際のUI（ナビゲーションリストなど）を意図通りに配置できるようになる。
  • Flexboxの入れ子構造を使い、複雑なレイアウトをシンプルに解決する感覚を掴む。

🏃 実践: 一緒に作ってみましょう！

Chapter 2で作成したプロフィールページは、まだレイアウトに古さが残っています。特に、SNSリンクが縦に並んでいる部分は改善の余地がありそうです。ここにFlexboxの力を注入して、洗練されたレイアウトを完成させましょう。

💭 実装の思考プロセス

今回修正するのは、主に「SNSリンク」の部分です。現状は<ul>のデフォルトの振る舞いで縦に並んでいますが、これを横並びにして、さらに見た目も整えたい。どうすれば実現できるか、思考プロセスを追ってみましょう。

このハンズオンのゴール:

  1. SNSリストの横並び: <ul>をFlexコンテナにして、<li>を横に並べる。
  2. 中央揃え: 並べたリストを、セクション内で水平方向の中央に配置する。
  3. 見た目の調整: 各リンクをボタンのようなデザインに装飾する。

💡 ポイント: 小さなコンポーネントからFlexboxを適用していくのがコツです。ページ全体を一度にFlexboxで制御しようとすると混乱します。まずは「ここの横並びを実現したい」という具体的な箇所を見つけて、その親要素にdisplay: flex;を適用することから始めましょう。

📝 ステップバイステップで実装

Step 1: SNSリストをFlexboxで横並びにする

まず、縦に並んでいるSNSリンクを横に並べます。

  • 何を考えているか: 「SNSリンクのリスト(<ul>)を横並びにしたい。ということは、<ul>をFlexコンテナにすれば、その子要素である<li>がFlexアイテムになって横に並ぶはずだ。やってみよう。」

1-1. index.htmlのSNSリスト部分の<ul>に、class="social-links"を追加します。これにより、CSSからこのリストを特定しやすくなります。

  • コマンド/コード (index.html):
    html <ul class="social-links"> <li><a href="#" target="_blank">GitHub</a></li> <li><a href="#" target="_blank">Twitter</a></li> <li><a href="#" target="_blank">ブログ</a></li> </ul>

1-2. style.cssに以下のコードを追加します。

  • コマンド/コード (style.css):
    css .social-links { display: flex; /* これで<li>が横に並ぶ */ justify-content: center; /* アイテムを中央に寄せる */ gap: 15px; /* アイテム間の隙間を15pxに */ list-style: none; /* 黒丸を削除 */ padding: 0; margin-top: 20px; }
  • コマンド解説/コードリーディング:
    • display: flex;: <ul>をFlexコンテナにします。これで子要素の<li>が横に並びます。
    • justify-content: center;: Flexアイテム（<li>たち）を、コンテナ（<ul>）の主軸（横方向）の中央に配置します。
    • gap: 15px;: Flexアイテムの間に15pxの隙間を設けます。marginで指定するより簡単です。
Step 2: リンクをボタン風にデザインする

横に並んだリンクを、よりクリックしやすいようにボタン風のデザインに装飾します。

  • 何を考えているか: 「ただのテキストリンクだと味気ない。<a>タグに背景色とpaddingを指定して、ボタンみたいに見せよう。マウスが乗ったら少し色が変わるエフェクトも付け加えたい。」

2-1. style.cssのli aセレクタを、.social-links aに修正し、以下のスタイルを追加します。（より具体的にセレクタを指定することで、他の箇所のリンクに影響を与えないようにします）

  • コマンド/コード (style.css):
    ```css
    / 既存の li a スタイルをこちらに統合・修正 /
    .social-links a {
    display: block; / paddingを有効にするためブロック要素に /
    padding: 10px 20px;
    background-color: #3498db; / 青い背景 /
    color: white; / 白い文字 /
    text-decoration: none;
    border-radius: 5px; / 少し角を丸める /
    transition: background-color 0.3s; / 背景色の変化を滑らかに /
    }

    / 既存の li a:hover スタイルをこちらに修正 /
    .social-links a:hover {
    background-color: #2980b9; / 少し濃い青に /
    text-decoration: none; / hover時も下線は不要 /
    }
    `` - **コマンド解説/コードリーディング:** -display: block;:タグはインライン要素なので、paddingやmarginを正しく適用するためにブロック要素に変換します。 -transition: background-color 0.3s;:background-color`プロパティが変更される際に、0.3秒かけて滑らかに変化するアニメーション効果を追加します。これにより、マウスオーバー時の色の変化がスムーズになります。

✨ 完成！

お疲れ様でした！ブラウザでページをリロードしてみてください。縦に並んでいたSNSリンクが、中央揃えの綺麗なボタン群に変わっているはずです。このように、既存のCSSに少し手を加えるだけで、Flexboxはレイアウトを劇的に改善してくれます。

このハンズオンを通じて、あなたは単に要素を横に並べるだけでなく、「どのように揃えるか」「どのように装飾するか」まで含めて、コンポーネントを自在にデザインする強力な武器を手に入れました。
